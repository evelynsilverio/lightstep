¿Qué es


Light


STEP?


LightStep es un sistema de iluminación inteligente que mejora la seguridad, eficiencia energética y comodidad en diversos entornos. Con LightStep puedes:

Cambiar el color de las luces a tu gusto.

Elegir efectos visuales (estático, radial, arcoíris).

Ver estadísticas sobre el tiempo de uso del sistema.

Programar el tiempo en que el sistema se apague o encienda.




Objetivo General

Desarrollar un sistema de iluminación inteligente, adaptable a escaleras, pasillos y diferentes espacios, que mejore la seguridad, optimice el consumo energético y brinde personalización mediante una aplicación móvil y plataforma web.

Objetivos Específicos

  • 1 Implementar sensores para detectar la presencia de personas y activar las luces automáticamente.
  • 2 Crear una aplicación móvil que permita al usuario personalizar el sistema.
  • 3 Diseñar una página web para visualizar las características del sistema.




01 Diseño Moderno y Estético. La integración de luces LED y el control por aplicación hacen que el sistema se vea elegante y futurista en cualquier espacio.

02 Automatización Inteligente. Gracias al uso de sensores de movimiento y control a distancia, el sistema opera sin necesidad de interacción manual constante.

03 Seguridad Mejorada. Al detectar movimiento y encender las luces automáticamente, reduce el riesgo de accidentes en escaleras y pasillos oscuros.
