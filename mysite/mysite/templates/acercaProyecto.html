¿Por qué es importante?

En un mundo donde la automatización y la eficiencia energética son clave, nuestro sistema de iluminación inteligente para escaleras ofrece:

    Seguridad: Reduce el riesgo de accidentes al iluminar automáticamente los escalones cuando detecta movimiento.

    Eficiencia energética: Las luces solo se encienden cuando es necesario.

    Innovación y confort: Transforma cualquier espacio en un entorno inteligente y moderno.

Tecnologías utilizadas

Para desarrollar un sistema eficiente y escalable, utilizamos herramientas modernas en hardware y software:

    Backend: Django para gestionar la lógica del sistema y la comunicación con la base de datos.

    Frontend: Tailwind CSS y Django Templates para una interfaz web intuitiva y moderna.

    Base de datos: Firebase para almacenar datos de configuración y uso del sistema.

    Microcontrolador: ESP32 para la gestión de sensores y luces LED.

    Aplicación móvil: Flutter para permitir el control remoto del sistema.

Nuestro enfoque

Nos diferenciamos por nuestra visión de crear un producto que no solo sea funcional, sino también adaptable y personalizable a las necesidades de cada usuario:

    Interfaz intuitiva y configuración sencilla a través de la app.

    Algoritmos inteligentes que optimizan el encendido y apagado.

    Ajuste de intensidad, colores y patrones de iluminación
