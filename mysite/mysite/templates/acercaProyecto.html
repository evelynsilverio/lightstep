¿Por qué es importante?

En un mundo donde la automatización y la eficiencia energética son clave, nuestro sistema de iluminación inteligente para escaleras ofrece:

    Seguridad: Reduce el riesgo de accidentes al iluminar automáticamente los escalones cuando detecta movimiento.

    Eficiencia energética: Las luces solo se encienden cuando es necesario.

    Innovación y confort: Transforma cualquier espacio en un entorno inteligente y moderno.

Tecnologías utilizadas

Para desarrollar un sistema eficiente y escalable, utilizamos herramientas modernas en hardware y software:

    Backend: Django para gestionar la lógica del sistema y la comunicación con la base de datos.

    Frontend: Tailwind CSS y Django Templates para una interfaz web intuitiva y moderna.

    Base de datos: Firebase para almacenar datos de configuración y uso del sistema.

    Microcontrolador: ESP32 para la gestión de sensores y luces LED.

    Aplicación móvil: Flutter para permitir el control remoto del sistema.

Nuestro enfoque

Nos diferenciamos por nuestra visión de crear un producto que no solo sea funcional, sino también adaptable y personalizable a las necesidades de cada usuario:

    Interfaz intuitiva y configuración sencilla a través de la app.

    Algoritmos inteligentes que optimizan el encendido y apagado.

    Ajuste de intensidad, colores y patrones de iluminación


Términos y Condiciones de Uso

1. Introducción
  • Esta sección establece los términos y condiciones del sistema de iluminación inteligente “LightStep” que es funcional para escaleras y pasillos, incluyendo funcionamiento con sensores, la aplicación móvil y la base de datos. Al utilizar este sistema, el usuario acepta los siguientes términos y condiciones.
2. Descripción del Servicio
  • El sistema de iluminación inteligente está diseñado para optimizar el uso de energía mediante el encendido y apagado automático de luces según la detección de movimiento. Además permite ajustes personalizados en la configuración de las luces a través de la aplicación móvil, proporciona información detallada a través de la aplicación web y gestiona datos en una base de datos.
3. Uso y Requisitos del Sistema
  • Los sensores detectan movimiento en escaleras, pasillos y otras áreas designadas, permitiendo activarse progresivamente las luces sin retrasos.
  • Las luces se apagan automáticamente tras un periodo de inactividad para reducir el consumo energético.
  • El sistema debe responder en menos de 5 segundos tras la detección de movimiento.
  • La aplicación web proporciona información detallada sobre el sistema, preguntas frecuentes y un formulario de contacto para soporte técnico.
  • La aplicación móvil permite la sincronización de configuraciones entre dispositivos mediante almacenamiento en la nube.
  • Todas las interfaces del sistema deben ser intuitivas y fáciles de usar.
4. Responsabilidades del Usuario
  • El usuario debe asegurarse de la instalación adecuada del sistema según las instrucciones proporcionadas.
  • Es responsabilidad del usuario mantener el sistema actualizado y realizar revisiones periódicas para garantizar su correcto funcionamiento.
  • El usuario no debe modificar o manipular los componentes del sistema sin autorización, ya que esto podría afectar su funcionamiento.
  • El sistema no garantiza un funcionamiento ininterrumpido ya que puede verse afectado por factores externos como cortes de energía, fallos en la conexión a internet, entre otros.
5. Privacidad y Seguridad
  • La información del usuario será utilizada únicamente para la configuración y personalización del sistema.
  • Se aplicarán medidas de seguridad para proteger los datos almacenados en la nube.
  • No se compartirá información personal con terceros sin el consentimiento expreso del usuario.
6. Garantía y Mantenimiento
  • El sistema cuenta con una garantía limitada contra defectos de fabricación por un período determinado de 365 días.
  • La garantía no cubre daños causados por uso indebido, instalación incorrecta o modificaciones no autorizadas.
  • Se recomienda realizar mantenimiento periódico del sistema, incluyendo limpieza de sensores y revisión de conexiones eléctricas, para garantizar su correcto funcionamiento.
7. Modificaciones y Actualizaciones
  • Los términos y condiciones pueden ser actualizados en cualquier momento para reflejar mejoras o cambios en el sistema. Se notificará a los usuarios sobre modificaciones relevantes.
8. Contacto y Soporte
  • Para consultas, soporte técnico o información adicional, el usuario puede comunicarse a través del formulario de contacto disponible en la aplicación web o en la aplicación móvil.
